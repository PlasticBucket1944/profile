ポートフォリオについて


概要

基本的なCRUDができるタスク管理アプリ

リンクはこちら(サーバー側の立ち上がりに30秒ほどかかります)→　Task App

TypeScriptを勉強する過程で作ったToDoクラスのアプリなので正直ポートフォリオというほどでもありませんが・・・

クライアント側はReact、サーバー側はNestjsでTypeScriptを利用して作成しました。
デプロイはクライアント側はGitHub Pages、サーバー側はrender.comの無料サーバー&DBを利用しています。

クライアント側 開発環境

typescript : 4.7.4
react : 18.2.0

リポジトリはこちら→　Github

サーバー側 開発環境

typescript : 4.3.5
nestjs : 8.2.6
PostgreSQL : 14.2
pg : 8.7.3
@nestjs/typeorm : 8.0.2
typeorm : 0.2.45

リポジトリはこちら→　Github

開発の動機、苦労した事とか
  • TypeScript勉強してなんか作りたくなった
  • rails、LaravelのフルスタックWebフレームワークは使ったことがあったが、最近主流のマイクロサービス形式でクラサバを分けたアプリを作りたくなった
  • Reactは昔チュートリアルをやっただけ、Nestjsは素人
  • 特にNestjsは日本語の情報が少なかったので苦労した、結局公式のドキュメントを読みつつUdemyの講座で勉強した
  • デプロイもHerokuにしようとしたら無料枠がなくなり、急遽render.comにすることに
  • さらに情報が少なくて苦労した、結局クライアント側(こっちはGithub Pages)もサーバー側も英語の記事を漁ってなんとかなった
  • 一通りデプロイまで経験できて良い機会だった、が、肝心のアプリの内容がしょぼい・・・(´・ω・｀)
